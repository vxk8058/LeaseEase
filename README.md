# LeaseEase
### AI-Powered Voice Assistant for Car Leasing

#Overview
### LeaseEase is an voice-based assistant designed to make the financial aspect of car shooping conversational, transparent, and accessible. 
Users can talk to LeaseEase and explore Toyota models and personalize car recommendation based on financial situations. 

## Key Features
- Voice Interaction
- Affordability Calculator
- Match recommendations
- Text to Speech
- Accesibility focus
